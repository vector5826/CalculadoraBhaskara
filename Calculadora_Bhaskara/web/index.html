//Desafio do tio! Usando o projeto da calculadora IMC como exemplo, implemente um novo projeto para uma pequena calculadora de Bhaskara (equações do segundo grau). Lembre-se que para calcular as raízes de uma eq. do segundo grau utilizando o método de Bhaskara, é preciso conhecer os três valores "a", "b" e "c" da própria equação. Portando... Dada uma eq. no formado ax2 + bx + c = 0 Calcular o Delta... Delta = b^2 - 4 * a * c E calcualr as raízes... Raíz Positiva = [(-b) + raiz(delta)] / (2 * a) Raíz Negativa = [(-b) - raiz(delta)] / (2 * a) Pontos importantes: - Se delta der negativo, não é possível calcular as raízes com este método (parar o programa) - "a" numca pode ser 0 Não vale nota.. apenas para brincar e treinar..
